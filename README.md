**Build Setup for VCPKG:**

```bash
 # CMake  with vcpkg
 cmake .. -DCMAKE_TOOLCHAIN_FILE=~/vcpkg/scripts/buildsystems/vcpkg.cmake
```

```bash
 # CMake  with vcpkg and emcmake
 emcmake cmake .. -DCMAKE_TOOLCHAIN_FILE=~/vcpkg/scripts/buildsystems/vcpkg.cmake
```
